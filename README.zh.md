## 💡 什么是 Project Alttl?  
Project Alttul 是一个开源项目，旨在帮助人们节省开支并提高生活效率。无论是通过自动化、节约成本的解决方案，还是智能理财工具，我们的目标都是让科技为您的钱包服务。  

## 🚀 Video2Text - 主要功能  
- AI驱动的视频/音频文本摘要  
- 多语言支持（英语、韩语、中文、日语）  
- 轻量级、用户友好的设计  

## 🖥️ 系统要求  
### ✅ **硬件要求**  
- **最低 GPU:** **6GB VRAM** 及以上的 NVIDIA 显卡  
- **推荐 GPU:** **8GB+ VRAM** 的 NVIDIA RTX 显卡  

### ✅ **软件要求**  
- **NVIDIA 显卡驱动**（最新版本）  
- **NVIDIA CUDA 1.18**  
- **Python 3.10**  
- **所需 Python 模块:** 见 `requirements.txt`
